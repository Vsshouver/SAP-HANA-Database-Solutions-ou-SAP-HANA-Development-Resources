Este repositório contém recursos, exemplos de código e melhores práticas para desenvolvimento e administração no SAP HANA, o banco de dados in-memory de alta performance da SAP. 

Modelos de dados e procedimentos SQLScript

Exemplos de aplicações com XS Advanced (XSA)

Scripts de administração e otimização

Integrações com outras soluções SAP

Projetos de análise de dados com SAP HANA

Principais Tópicos Abordados
✔️ Modelagem de dados em SAP HANA
✔️ Desenvolvimento de procedures e funções em SQLScript
✔️ Configuração e otimização de performance
✔️ SAP HANA XS Advanced (XSA) e aplicações nativas
✔️ Integração com SAP BW/4HANA e outros sistemas SAP
✔️ Machine Learning no SAP HANA

Para Quem é Útil
Desenvolvedores SAP HANA

Administradores de banco de dados

Arquitetos de soluções SAP

Consultores de BI e Analytics

Estudantes de tecnologias SAP
